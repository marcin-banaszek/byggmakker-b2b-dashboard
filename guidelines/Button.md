# Button

## Use

- `variant="default"` for the main action in a section.
- `variant="secondary"` for supporting actions.
- `variant="outline"` or `variant="ghost"` for low-emphasis actions.

## Rules

- Keep labels short and action-oriented.
- Avoid multiple primary buttons in one compact action area.
- Keep default focus ring visible.

## Sizes

- `default` for common actions.
- `sm` for dense tables/cards.
- `lg` for hero/landing call-to-actions.
- `icon` for icon-only controls with `aria-label`.
